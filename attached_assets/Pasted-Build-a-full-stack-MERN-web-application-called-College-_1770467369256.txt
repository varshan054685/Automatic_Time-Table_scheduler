Build a full-stack MERN web application called
"College Automatic Timetable Scheduler".

TECH STACK REQUIREMENTS:
- OS: Ubuntu Linux
- Frontend: React (JavaScript only, no TypeScript)
- Styling: Tailwind CSS
- Backend: Node.js + Express (JavaScript)
- Database: MongoDB (local MongoDB or MongoDB Atlas)
- Stack: MERN
- Package manager: npm

TARGET USERS:
- College staff and administrators
- Application should run on localhost
- Simple UI suitable for non-technical users

FUNCTIONAL REQUIREMENTS:

1. AUTHENTICATION
- Staff login and registration
- Password hashing using bcrypt
- JWT-based authentication
- Protected routes for staff/admin

2. MASTER DATA MANAGEMENT (CRUD)
Create UI screens and REST APIs for:
- Departments
- Subjects (subject code, name, weekly hours)
- Faculty (name, department, availability)
- Classrooms (room number, capacity)
- Classes / Sections (year, semester)
- Time slots (days and periods)

3. AUTOMATIC TIMETABLE GENERATION
Implement an automatic timetable scheduler that:
- Prevents faculty time conflicts
- Prevents classroom conflicts
- Assigns subjects based on weekly hours
- Respects faculty availability
- Generates timetable per class/section
- Uses a rule-based or greedy scheduling algorithm
- Returns timetable data as JSON

4. TIMETABLE DISPLAY
- Display timetable in grid format (days vs periods)
- Views:
  - By Class
  - By Faculty
- Color-coded subjects
- Responsive UI using Tailwind CSS

5. EXPORT FEATURES
- Export timetable as PDF
- Export timetable as Excel
- Print-friendly layout

PROJECT STRUCTURE:

Backend:
- Express REST API
- MVC folder structure
- MongoDB models using Mongoose
- Separate service for timetable generation logic

Frontend:
- React (Vite preferred)
- Tailwind CSS configured
- Pages:
  - Login
  - Dashboard
  - Data Management
  - Generate Timetable
  - View Timetable
- Axios for API calls

NON-FUNCTIONAL REQUIREMENTS:
- Clean and readable JavaScript code
- Proper comments for scheduling logic
- No TypeScript files
- No cloud-only dependencies
- App must run locally on Ubuntu

DELIVERABLES:
- Complete backend code
- Complete frontend code
- Clear README with setup instructions
- Commands to run frontend and backend locally
